# Start Here HTML Pages

Add the complete `start-here` folder to the repository root.

```text
funding-workflow-builder/
├── index.html
├── styles.css
├── script.js
├── embed.html
├── assets/
└── start-here/
    ├── getting-started.html
    └── privacy-input-guide.html
```

The pages reuse the existing root `styles.css`, `script.js`, favicon, and Open Graph asset. Page-specific styling and copy-to-clipboard behavior are included directly in each HTML file.

Production URLs after deployment:

- https://funding-workflow-builder.vercel.app/start-here/getting-started.html
- https://funding-workflow-builder.vercel.app/start-here/privacy-input-guide.html
